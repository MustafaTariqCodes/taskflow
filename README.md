# TaskFlow

A full-stack task management web application for managing projects and tasks.

## Features

- Create Projects
- Delete Projects
- Create Tasks
- Delete Tasks
- Progress Tracking
- Responsive UI
- REST API

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python
- FastAPI

### Database
- MySQL

### Deployment
- Railway (Backend)
- Vercel (Frontend)
